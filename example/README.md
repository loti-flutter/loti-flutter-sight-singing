# Loti 音乐视唱插件 · 示例 App

本目录为 **Loti Flutter 音乐视唱插件** 的示例工程，用于在本地预览产品介绍页、三档定价与收款信息。

> 当前示例 App 为**产品预览占位**，展示插件接入状态与商务信息。完整 MIDI 视唱、实时打分等能力需购买授权后交付。

---

## 示例 App 包含什么

- 产品简介与核心能力说明
- 三档版本定价（测试版 / 年度授权 / 源码交付）
- 嘉联支付收款码与付款备注说明
- 调用占位 API `LotiSightSingingPlugin.getStatus()` 展示插件版本

---

## 运行方式

在项目根目录克隆仓库后：

```bash
cd example
flutter pub get
flutter run
```

支持 iOS、Android；Web 可按 Flutter 常规方式运行（以本机环境为准）。
